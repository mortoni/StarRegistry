# Post New Block

Get Star block by wallet address (blockchain identity).

**URL** : `/stars/address/:[ADDRESS]`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

## Success Response

**Code** : `200 OK`

**Content examples**

`address: 142BDCeSGbXjWKaAnYXbMpZ6sbrSAo3DpZ`

## Notes

* If the parameter `address` is empty it does return an error.
* The response includes entire star block contents along with the addition of star story decoded to ASCII.
* This endpoint response contained a list of Stars because of one wallet address can be used to register multiple Stars.
